                                        *** Python Data Types ***

# Data Types.
- String (str). 
> x = "Hello World"
- Integer (int).
> x = 20
- Float (float).
> x = 20.5 	
- Complex (complex).
> x = 1j
- List (list).
> x = ["apple", "banana", "cherry"]
- Tuple (tuple). 	
> x = ("apple", "banana", "cherry")
- Range (range). 	
> x = range(6)
- Dictionary (dict).
> x = {"name" : "John", "age" : 36}
- Set (set). 	
> x = {"apple", "banana", "cherry"}
- Frozen Set (frozenset). 	
> x = frozenset({"apple", "banana", "cherry"})
- Bool (bool).	
> x = True
- Bytes (bytes).
> x = b"Hello" 	
- Byte Array (bytearray).
> x = bytearray(5) 	
- Memory View (memoryview).
> x = memoryview(bytes(5)) 	
- None Type (nonetype).
> x = None
